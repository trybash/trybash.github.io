Cheat Sheet

The shell is dark and full of terrors. Take this:

ls

List all files and folders in the current directory

ls foldername

List all files and folders in the folder named foldername

ls -l

List all files and folders with addictional information

ls -a

List even hidden files or folders!

ls -R

List files,folders and all the subfolders

mkdir name

Create a folder named name

rm filename

Remove the file named filename

and so on
